# 2026 世界杯预测分析系统

一个本地网页应用，用于分析和预测 2026 美加墨世界杯的比赛结果。

## 功能特点

### 🏠 首页
- 世界杯概览和统计信息
- 热门比赛快速访问
- 系统功能介绍

### ⚽ 球队分析
- 48 支参赛球队详细信息
- 按小组（A-L 组）筛选
- 球队实力、身价、排名等数据
- 核心优势和短板分析
- 近期表现数据

### 📊 比赛预测
- 所有小组赛预测结果
- 按小组和轮次筛选
- 胜平负概率分析
- 预测比分和进球数
- 比赛看点和分析

### 🔄 球队对比
- 选择任意两支球队进行对比
- 多维度数据对比（排名、身价、胜率等）
- 可视化对比图表
- 历史交锋分析

### 👥 球员数据
- 球员详细信息（身高、体重、年龄等）
- 按球队和位置筛选
- 球员搜索功能
- 赛季数据统计

## 快速开始

### 方法一：使用启动脚本（推荐）

1. 双击运行 `start.bat`
2. 在浏览器中访问 `http://localhost:8080`

### 方法二：手动启动

1. 打开命令行（CMD 或 PowerShell）
2. 进入 `webapp` 目录
3. 运行以下命令：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080

# 或使用 PHP
php -S localhost:8080
```

4. 在浏览器中访问 `http://localhost:8080`
